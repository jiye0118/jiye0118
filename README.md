### 🛠 Kidztime CS Tool — Shopee CS 응대 도구
Shopee 셀러의 반복적인 고객 응대를 줄이기 위해 직접 기획·제작·배포한 사내 도구.

- **스택** — Next.js · Tailwind CSS · shadcn/ui · Claude API · Vercel
- 🔗 [Live Demo](https://kidztime-cs-tool.vercel.app) · [Repository](https://github.com/jiye0118/kidztime-cs-tool)
